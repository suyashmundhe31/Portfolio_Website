# Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and professional experience.

## Features

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional interface with smooth animations
- **Project Showcase** - Detailed project presentations with live demos and source code links
- **About Section** - Professional background, skills, and achievements
- **Contact Integration** - Easy ways to get in touch and connect
- **Performance Optimized** - Fast loading times and smooth user experience

## Built With

- **Frontend Framework** - React with TypeScript
- **Styling** - CSS3 with modern design principles
- **Build Tool** - Vite for fast development and optimized builds
- **Code Quality** - ESLint for consistent code standards
- **Package Management** - npm/yarn for dependency management

## Project Structure

```
src/
├── assets/          # Static assets (images, documents)
├── components/      # Reusable React components
│   ├── Header.tsx   # Navigation header
│   ├── Footer.tsx   # Site footer
│   └── Layout.tsx   # Main layout wrapper
└── pages/           # Main page components
    ├── About.tsx    # About me section
    ├── Achievements.tsx # Professional achievements
    ├── Contact.tsx  # Contact information
    ├── Experience.tsx # Work experience
    ├── Home.tsx     # Landing page
    └── Projects.tsx # Project portfolio
```
